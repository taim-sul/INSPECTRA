# Din UdlejningsApp

PWA-klar bil-udlejning og biltjek app.

## Upload til GitHub Pages

1. Opret nyt GitHub repository (Public)
2. Upload alle filer i denne mappe
3. Settings → Pages → Branch: main → Save
4. Din app er live på: `https://[dit-navn].github.io/[repo-navn]`

## Filer
- `index.html` — selve appen
- `manifest.json` — PWA konfiguration
- `sw.js` — Service Worker (offline support)
- `icon-192.png` — app ikon (lille)
- `icon-512.png` — app ikon (stort)
